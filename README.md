# Portfolio-People-Analytics
Este reporte fue diseñado para transformar datos de Recursos Humanos en indicadores estratégicos (KPIs) mediante el uso de Power BI. El objetivo principal es analizar la salud organizacional a través de la retención de talento y el análisis salarial.


Dashboard de People Analytics: Análisis de Retención y Compensación
Contexto del Proyecto: Como profesional formada en Historia, mi enfoque siempre ha sido entender procesos complejos a través del análisis crítico de evidencia. Este proyecto representa mi transición hacia el área de Recursos Humanos, utilizando Power BI para transformar datos crudos en decisiones estratégicas para la gestión de personas.

Objetivos del Dashboard:

Monitoreo de Plantilla: Visualización clara del Headcount actual (415 Empleados Activos).

Gestión de Bajas: Seguimiento del total de salidas (85 Bajas) para identificar áreas de riesgo en la retención.

Análisis Salarial: Evaluación de la inversión en talento mediante el Sueldo Promedio por Departamento.

Interactividad: Filtros dinámicos que permiten una exploración detallada por diversas variables organizacionales.

Habilidades Técnicas Demostradas:

DAX: Creación de medidas personalizadas para cálculos de rotación y promedios.

Power Query: Limpieza y transformación de datos, incluyendo el manejo de valores nulos en fechas de salida.

Visualización de Datos: Implementación de un diseño "Modern Dark" enfocado en la legibilidad ejecutiva.
